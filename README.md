# Plain Nuxt Application

This is a plain Nuxt 4 application with no instrumentation.

## Features

- Basic Nuxt 4 setup
- Simple hash API endpoint
- No monitoring or tracing

## Running

```bash
npm install
npm run dev
```

Navigate to http://localhost:3000
